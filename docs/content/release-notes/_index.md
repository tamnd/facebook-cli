Notable changes per release. fb follows semantic versioning; until 1.0 the
command surface may still shift between minor versions.

- [v0.1.2](/release-notes/v0-1-2/): public groups read in full, with posts and comments.
- [v0.1.1](/release-notes/v0-1-1/): reads Facebook without a cookie, plus a new archive command.
- [v0.1.0](/release-notes/v0-1-0/): the first release.
