Each guide is a focused walkthrough of one part of fb. Start with
[authentication](/guides/authentication/) if you have not set a session yet,
since it is what unlocks everything else.

- [Authentication](/guides/authentication/): cookie formats and what a session unlocks.
- [Pages and profiles](/guides/pages-and-profiles/): resolve entities and stream feeds.
- [Posts and comments](/guides/posts-and-comments/): one story, its thread, and reactions.
- [Media](/guides/media/): photos, videos, reels, and events.
- [Search and discovery](/guides/search-and-discovery/): find entities and classify ids.
- [Datasets](/guides/datasets/): seed, crawl, and query a local SQLite store.
